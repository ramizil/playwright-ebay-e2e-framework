4. Step Dependencies (auto-skip on failure)

How it works: The pytest_runtest_setup hook in conftest.py checks if any earlier step in the same class has failed. If yes, it calls pytest.skip("Previous step failed"). This is tracked per parametrised variant via _failed_class_nodes on the pytest config object.
